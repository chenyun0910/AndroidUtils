# AndroidUtils
android一些工具类

## Quick start

Add jcenter to your project's gradle file

```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

####Gradle

```
implementation 'com.github.chenyun0910:AndroidUtils:1.0'
```

#### Maven

```
<dependency>
   <groupId>com.github.chenyun0910</groupId>
   <artifactId>AndroidUtils</artifactId>
   <version>Tag</version>
</dependency>
```
